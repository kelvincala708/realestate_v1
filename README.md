# Way Home Real Estate - Luxury Real Estate Website

A premium one-page luxury real estate website mockup for Way Home Real Estate.

## Project Structure

```
realestate_v1/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles
├── js/
│   └── script.js       # All JavaScript functionality
├── assets/
│   └── images/         # Image assets (add your images here)
└── README.md           # This file
```

## Adding Images

Place your images in the `assets/images/` directory with the following names:

### Logo
- `image(1).png` - Company logo

### Hero Section
- `Heroimage(1).avif` - Main hero image (large luxury villa, landscape orientation)

### Services Section
- `serviceimage(1).avif` - Service section image (luxury interior or architectural detail)

### Final Banner
- `lastbanner(1).avif` - Final CTA banner image (luxury interior)

## Viewing the Website

1. Open `index.html` in a web browser
2. For best results, use a local development server:
   - VS Code: Use the "Live Server" extension
   - Or run: `npx serve` in the project directory

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-up effects on scroll, staggered reveals
- **Interactive Elements**: Hover effects, smooth scrolling navigation
- **Contact Form**: Functional form with validation
- **Mobile Menu**: Hamburger menu for mobile devices

## Notes

- The website uses Google Fonts (Playfair Display and Manrope)
- Images should be high-quality luxury real estate photos
- For production, optimize images for web (compress, use WebP format)
- The contact form currently shows an alert; connect to a backend for real submissions
